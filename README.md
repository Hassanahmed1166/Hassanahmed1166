## üíª **Projects**

### **DeepCAN: Intrusion Detection System for Controller Area Network**
*Tools: Kaggle, Python, Keras, TensorFlow, Pandas, NumPy, Scikit-learn, Matplotlib*
- Developed a deep learning-based Intrusion Detection System (IDS) for Controller Area Network (CAN) to enhance security of In-Vehicle Networks (IVNs) in Autonomous Vehicles
- Achieved **94.52% classification accuracy**, outperforming existing methods in accuracy and F1-score
- Detects and classifies impersonation, fuzzy, and denial-of-service (DoS) attacks
- Applied Explainable AI (XAI) methods using LIME for interpretability in IDS decision-making

### **AgriGuard: Pest Detection and Classification**
*Tools: Kaggle, Python, NumPy, Pandas, TensorFlow, Scikit-learn, Matplotlib, OpenCV, Seaborn*
- Developed a pest image classification system for agricultural pest detection, enabling early intervention
- Implemented Deep CNN achieving **90.00% accuracy**, improving detection efficiency
- Created a deep learning pipeline with data augmentation and transfer learning for improved generalization
- Applied confusion matrix and classification reports to optimize the classification process

### **Urdu to Roman Urdu Transliteration System**
*Tools: Kaggle, Python, PyTorch, NumPy, Pandas, NLTK, JiWER, Matplotlib*
- Developed a Seq2Seq-based transliteration system to convert Urdu text into Roman Urdu
- Implemented BiLSTM encoder and multi-layer LSTM decoder architecture for character-level dependencies
- Achieved **BLEU score of 0.6943**, **perplexity of 1.1687**, and **character error rate of 0.0997**
- Trained on a custom parallel Urdu‚ÄìRoman Urdu dataset with tokenization and teacher forcing

### **Breast Cancer Prediction using Machine Learning**
*Tools: Python, Pandas, NumPy, Seaborn, Matplotlib, Scikit-Learn*
- Developed breast cancer prediction model achieving **94-98% accuracy**
- Implemented data preprocessing and feature selection for thousands of patient records
- Created correlation heatmaps and distribution plots for enhanced clinical interpretability
- Built a classification pipeline for integration with hospital decision-support systems
